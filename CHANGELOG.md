## [0.1.0] - 2025-11-23

### Added
- ‚ú® Initial release
- ‚ú® Text selection and annotation creation
- ‚ú® Bidirectional anchoring between annotations and text
- ‚ú® `<mark_N>` tag system for data persistence
- ‚ú® Tag echo functionality for saved annotations
- ‚ú® Context-based positioning for duplicate text (using split method)
- ‚ú® Modern UI with professional styling
- ‚ú® Full TypeScript support
- ‚ú® Comprehensive documentation

### Features
- Select any text in Markdown to add annotations
- Click annotation cards to navigate to text
- Click highlighted text to navigate to annotations
- Edit and delete annotations
- Automatic ID generation
- Support for controlled and uncontrolled modes
- Precise duplicate text handling with context-based positioning
