# API 文档

本文档详细说明了 `markdown-annotation-kit` 的所有 API。

## 导出

### MarkdownAnnotator

主要的批注组件。

```typescript
import { MarkdownAnnotator } from "markdown-annotation-kit";
```

### 类型导出

```typescript
import type { MarkdownAnnotatorProps, AnnotationItem } from "markdown-annotation-kit";
```

### 工具函数导出

```typescript
import {
  exportAnnotationData,
  importAnnotationData,
  createDebouncedPersistence,
} from "markdown-annotation-kit";
```
