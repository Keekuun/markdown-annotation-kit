# 介绍

Markdown Annotation Kit 是一个生产级的 React 组件，用于为 Markdown 文档添加选择式批注和双向锚定功能。

## 特性

- 🎯 **精确文本选择** - 选择 Markdown 中的任何文本添加批注，支持上下文定位，准确处理重复文本
- 🔗 **双向锚定** - 点击批注卡片跳转到文本，点击高亮文本跳转到批注
- 🏷️ **标签系统** - 使用 `<mark_N></mark_N>` 标签进行数据持久化
- 🔄 **标签回显** - 自动识别并显示已保存的批注
- 🎨 **现代化 UI** - 专业的界面设计，流畅的动画和过渡效果
- 📦 **TypeScript** - 完整的 TypeScript 支持
- 🎛️ **受控/非受控** - 支持受控和非受控两种模式
- ⚡ **轻量级** - 最小化依赖
- ♿ **可访问性** - 支持键盘导航和 ARIA 属性
- 🎨 **可定制** - 通过 CSS 变量和类名轻松定制样式
- 🧪 **测试完善** - 使用 Vitest 提供全面的测试覆盖
- 📚 **文档完善** - 完整的 API 文档和示例

## 适用场景

- 📝 文档协作和审阅
- 💬 代码注释和讨论
- 📖 在线文档系统
- 🎓 教育平台
- 📋 内容管理系统

## 浏览器支持

- Chrome (最新版)
- Firefox (最新版)
- Safari (最新版)
- Edge (最新版)

需要支持 ES6+ 和以下 API：
- `getSelection` API
- `scrollIntoView` API
