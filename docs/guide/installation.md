# 安装

## 使用 npm

```bash
npm install markdown-annotation-kit
```

## 使用 pnpm

```bash
pnpm add markdown-annotation-kit
```

## 使用 yarn

```bash
yarn add markdown-annotation-kit
```

## 依赖要求

### Peer Dependencies

此包需要 React 17 或更高版本：

```json
{
  "react": ">=17",
  "react-dom": ">=17"
}
```

### 样式导入

组件需要导入样式文件：

```tsx
import "markdown-annotation-kit/styles.css";
```

或者在你的 CSS 入口文件中：

```css
@import "markdown-annotation-kit/styles.css";
```

## 类型定义

组件使用 TypeScript 编写，并提供了完整的类型定义。如果你使用 TypeScript，类型会自动包含在包中。
